# SulowCode RPA 设计器

一个基于 Go + Wails 的 RPA (机器人流程自动化) 低代码设计器，支持可视化流程设计和 C# 代码生成。

## 功能特性

- 🎨 **可视化设计**: 拖拽式组件设计 RPA 流程
- 🔧 **代码生成**: 自动生成可执行的 C# 代码
- 🐛 **调试支持**: 内置调试器，支持断点和单步执行
- 📦 **单文件部署**: 打包为独立的 EXE 文件，无需额外依赖
- 🖥️ **桌面应用**: 内嵌 Web 界面的桌面应用程序

## 技术栈

- **后端**: Go 1.23 + Wails v2.11.0
- **前端**: React + Vite
- **目标平台**: Windows (可扩展到其他平台)
- **代码生成**: C# (.NET)

## 快速开始

### 运行应用

1. 直接运行构建好的可执行文件:
   ```
   SulowCode.exe
   ```

2. 或者从源码构建:
   ```bash
   # 安装依赖
   go mod tidy
   cd frontend && npm install
   
   # 构建应用
   wails build
   ```

### 使用方法

1. **添加组件**: 点击工具栏中的组件按钮添加 RPA 节点
   - 点击组件: 模拟鼠标点击操作
   - 输入组件: 模拟键盘输入操作  
   - 等待组件: 添加延时等待

2. **生成代码**: 点击"生成代码"按钮查看生成的 C# 代码

3. **运行流程**: 点击"运行流程"按钮执行完整的工作流

4. **清空画布**: 点击"清空画布"重新开始设计

## 项目结构

```
SulowCode/
├── app.go              # 主应用逻辑
├── main.go             # 程序入口
├── wails.json          # Wails 配置
├── frontend/           # 前端代码
│   ├── src/
│   │   ├── App.jsx     # 主界面组件
│   │   └── App.css     # 样式文件
│   └── package.json
└── build/
    └── bin/
        └── SulowCode.exe  # 构建输出
```
