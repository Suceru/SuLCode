### 🎨 界面功能

#### 📋 组件面板（左侧）
- **折叠/展开**: 点击左上角 ◀ 折叠，点击左侧 ▶ 展开
- **分类管理**: 基础操作和流程控制分类
- **一键添加**: 点击组件即可添加到画布

#### 🖥️ 画布区域（中央）
- **无限画布**: 支持滚动查看更大区域
- **网格背景**: 点状网格帮助对齐节点
- **拖拽功能**: 节点可自由拖拽调整位置

#### 💻 代码面板（右侧）
- **按需显示**: 点击"生成代码"后自动显示
- **可折叠**: 点击右上角 ▶ 折叠面板
- **重新展开**: 折叠后点击右侧 ◀ 重新展开
- **深色主题**: 专业的代码显示界面

## 操作指南

### 1. 添加和管理组件
```
左侧组件面板 → 点击组件 → 自动添加到画布
支持组件：
👆 点击操作 - 模拟鼠标点击
⌨️ 输入文本 - 模拟键盘输入  
⏱️ 等待延时 - 添加时间延迟
```

### 2. 拖拽调整节点
```
操作步骤：
1. 鼠标悬停在节点上
2. 按住鼠标左键
3. 拖拽到目标位置
4. 释放鼠标完成移动

视觉反馈：
- 拖拽时节点会旋转和放大阴影
- 实时显示新位置
- 防止拖拽到画布外部
```

### 3. 代码生成和查看
```
生成代码：
顶部工具栏 → 📄 生成代码 → 右侧显示代码面板

管理代码面板：
- 自动显示：生成代码后自动展开
- 手动折叠：点击面板右上角 ▶
- 重新展开：点击右侧边缘 ◀ 按钮
- 关闭代码：折叠面板或点击清空按钮
```

### 4. 界面布局控制
```
组件面板控制：
- 折叠：点击面板右上角 ◀
- 展开：点击左侧边缘 ▶ 按钮

代码面板控制：
- 折叠：点击面板右上角 ▶  
- 展开：点击右侧边缘 ◀ 按钮

画布区域：
- 自动适应：根据面板状态自动调整大小
- 滚动查看：支持水平和垂直滚动
```

## 快捷操作

### 鼠标操作
- **左键单击**: 选择组件、按钮操作
- **左键拖拽**: 移动节点位置
- **右键单击**: 删除节点（点击节点上的 × 按钮）

### 面板操作
- **组件面板**: 左侧可折叠，不影响画布使用
- **代码面板**: 右侧可折叠，按需显示代码
- **画布区域**: 自适应大小，支持滚动

### 状态反馈
- **实时状态**: 底部状态栏显示当前操作状态
- **节点计数**: 实时显示当前画布节点数量
- **视觉反馈**: 所有交互都有动画和状态提示

## 技术特点

- **响应式布局**: 自适应不同屏幕尺寸
- **流畅动画**: 优化的CSS动画效果
- **内存优化**: 高效的状态管理
- **跨平台**: 基于Web技术，易于扩展
