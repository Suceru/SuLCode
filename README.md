# SuLCode
A professional low-code RPA designer for C# automation. Built with Go, Wails, and React. Features visual workflow editing, Roslyn-based code generation, and step-by-step debugging.

初始化项目:

Bash  
src\suceru\SuLCode>  
wails init -n SulowCode -t react  

cd core/frontend  
npm install reactflow  

cd ../../  
cd SulowCode  
wails dev  
wails build
